*****Hex game guide*******

Firstly Hex Game asks the user how many games he/she wants to open.
At least 5 games can be opened.

From the menu on the screen, 
the user can start a new game or select active games and start playing.

The user can compare the games from the menu after exiting the game.

The 3rd option in the menu compares the two selected games.
This option uses bool function that in the class.

The 4rd option in the menu compares all games and prints all information on the screen.
This option uses static int to find the total number of marks.
